# Reference archive

Offline copies of the external sources behind Winchy's winch-launch
understanding (`docs/winch_launch_physics.md` and the project's literature
notes), kept here so the project still has them if the originals move or are
taken offline — several already gate or block direct download (see notes).
Retrieved **2026-06-14**.

**Copyright:** each file is the work of its respective author/publisher,
included **unmodified** for preservation and reference only. These works are
**not** covered by Winchy's GPL-3.0 license; their own terms apply — follow the
source for licensing/reuse. (Same footing as the vendored drivers in
`firmware/rope/lib/`: third-party, retained as-is.)
